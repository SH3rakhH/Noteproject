# Changelog
Все заметные изменения этого проекта документируются в этом файле.  
Формат основан на принципах Keep a Changelog и проект следует Semantic Versioning.[web:338][web:392]


## [1.4.0] - 2026-07-22
### Added
- Брендинг приложения: переименование в «Zeero заметки» во всех заголовках и диалогах.[web:324][web:394]
- Отдельный диалог авторизации с названием «Zeero заметки».
- Экспорт одной заметки в `.txt` и `.md`.
- Пагинация списка заметок: кнопки «Предыдущая/Следующая» и отображение текущей страницы.[web:324][web:394]
- Экспорт всех заметок в один HTML‑файл с оформлением и метаданными (категория, даты, теги).
- Базовая поддержка категорий/тем для заметок.
- Поиск по тегам с отдельным полем ввода и фильтрацией списка заметок.[web:324][web:394]
- Массовое удаление заметок по выбранным тегам через диалог выбора тегов.[web:324]
- Улучшенный предпросмотр: всплывающий tooltip с заголовком, категорией и первыми строками текста заметки.[web:310]
- Облако тегов и статистика по тегам.
- Настраиваемый размер страницы для пагинации.
- Экспорт заметок в PDF через встроенный конвертер.

### Changed
- Обновлено значение `APP_VERSION` до `1.4.0` в соответствии с Semantic Versioning.[web:316]
- Переформулированы сообщения статуса и уведомлений под бренд Zeero.
- Структура окна: левая колонка для списка и фильтров, правая — для редактирования заметки.
- Меню File дополнено пунктами экспорта заметок.
- Обновлён статус‑бар: отображение текущей версии приложения и краткого состояния.
- Текстовые сообщения и диалоги приведены к единому стилю.
- Улучшен интерфейс предпросмотра заметок.
- Оптимизированы запросы к базе данных при большом количестве заметок.


### Fixed
- Небольшие текстовые правки и единообразие названий в интерфейсе.
- Стабильность отображения истории версий заметок.
- Исправления сохранения дат создания/изменения заметки.
- Стабильность работы при большом количестве записей.
- Корректная работа пагинации при фильтрации по категории.
- Исправлены редкие ошибки при загрузке превью заметки.
- Мелкие исправления UI и обработка ошибок.
