# Notes App (Python, Tkinter, SQLite)

Простое десктопное приложение для заметок на Python с графическим интерфейсом на Tkinter и базой данных SQLite.  
Позволяет создавать, редактировать и удалять заметки, хранить их в базе, видеть даты создания и изменения, а также получать всплывающие уведомления при операциях.

## Функционал

- Графический интерфейс на Tkinter и ttk (кнопки, поля ввода, список заметок).  
- Хранение заметок в локальной базе SQLite (`notes.db`).  
- Поля заметки:
  - заголовок,
  - текст,
  - дата создания (`created_at`),
  - дата изменения (`updated_at`).  
- Список заметок слева, выбранная заметка (текст + даты) справа.  
- Кнопки:
  - **Новая** — создание новой заметки,
  - **Сохранить** — сохранение / обновление заметки,
  - **Удалить** — удаление заметки с подтверждением.  
- Всплывающие «toast»‑уведомления при сохранении, обновлении и удалении заметок (не блокируют интерфейс).  
- Статус‑бар внизу с текстом последней операции и версией приложения.  
- Поддержка пользовательской иконки окна (`icon.ico` или `icon.png`).

## Установка

1. Установите **Python 3.x** (если ещё не установлен).  
2. Склонируйте репозиторий или скачайте архив:

   ```bash
   git clone https://github.com/SH3rakhH/note-app-tkinter.git
   cd note-app-tkinter
   ```

3. Убедитесь, что в папке есть файл приложения, например:

   - `note_app.py` (основной скрипт).

4. Дополнительно (по желанию) положите рядом с `.py` файлом иконку:

   - `icon.ico` — для Windows (предпочтительно),
   - или `icon.png` — для других платформ.

Модули `tkinter` и `sqlite3` входят в стандартную библиотеку Python, никаких дополнительных зависимостей устанавливать не нужно.

## Запуск

В папке проекта выполните:

### Windows

```bash
python note_app.py
```

или, если установлено несколько версий Python:

```bash
py note_app.py
```

### Linux / macOS

```bash
python3 note_app.py
```

При первом запуске автоматически создаётся файл базы данных `notes.db` с таблицей `notes`.

## Использование

- В левой части окна отображается список заметок по заголовкам.  
- Нажмите **«Новая»**, чтобы очистить поля и начать создание новой заметки.  
- Введите заголовок в поле **«Заголовок»**.  
- Введите текст заметки в поле **«Текст заметки»**.  
- Нажмите **«Сохранить»**:
  - если заметка новая — она будет добавлена в базу,
  - если заметка уже выбрана — её данные обновятся.  
- После сохранения появится небольшое всплывающее уведомление, а статус‑бар обновится сообщением об операции.  
- Выберите любую заметку из списка слева, чтобы:
  - посмотреть/изменить её текст,
  - увидеть даты **создания** и **последнего изменения**.  
- Нажмите **«Удалить»**, чтобы удалить выбранную заметку (перед удалением появится окно с подтверждением).  
- Внизу окна отображает��я строка статуса вида:

  ```text
  Заметка создана | Версия: 1.3.0 (SQLite + ttk + dates + icon)
  ```

## Структура БД

Используется одна таблица `notes` со следующими полями:

- `id` – `INTEGER PRIMARY KEY AUTOINCREMENT`  
- `title` – `TEXT NOT NULL` — заголовок заметки  
- `content` – `TEXT NOT NULL` — текст заметки  
- `created_at` – `TEXT NOT NULL DEFAULT current_timestamp` — дата/время создания (UTC)[web:21][web:55]  
- `updated_at` – `TEXT NOT NULL DEFAULT current_timestamp` — дата/время последнего изменения (UTC)[web:21][web:57]  

При создании заметки `created_at` и `updated_at` заполняются автоматически.  
При обновлении заметки поле `updated_at` устанавливается в текущий момент.

## Иконка приложения

Приложение пытается загрузить иконку из файлов:

1. `icon.ico` в папке с `note_app.py`:
   - используется через `root.iconbitmap(path)` (рекомендуется для Windows).[web:66][web:72]
2. Если `.ico` недоступен, используется `icon.png`:
   - через `PhotoImage` и `root.iconphoto(True, image)` — подходит для разных платформ.[web:68][web:72]

Если ни один файл не найден или произошла ошибка, будет использована стандартная иконка Tkinter.
