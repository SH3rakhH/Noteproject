# Zeero заметки

Zeero заметки — настольное приложение для ведения заметок студента на Python, Tkinter и ttkbootstrap.  
Поддерживает категории и теги, быстрый поиск (FTS5), пагинацию, историю версий, экспорт и сборку в `.exe`.

## Возможности

- Современный интерфейс на Tkinter + ttkbootstrap (light/dark темы, иконки на кнопках).[web:207][web:435]
- PIN-авторизация перед доступом к заметкам.
- Создание, редактирование и удаление заметок.
- Категории (например, «Математика», «История», «Программирование», «TO-DO»).
- Теги с привязкой к заметкам и облаком тегов в будущем.
- Поиск:
  - FTS5-поиск по заголовку и содержимому заметок.
  - Поиск по тегам с фильтрацией по категории.
- Пагинация списка заметок при большом количестве записей.
- История версий заметок (просмотр и откат содержимого).
- Улучшенный превью: tooltip с заголовком, категорией и первыми строками текста.
- Экспорт:
  - одной заметки в `.txt` и `.md`;
  - всех заметок в единый HTML‑файл;
  - всей базы данных SQLite (`notes.db`).
- Сборка в `.exe` через PyInstaller и создание установщика через Inno Setup.[web:343][web:427][web:431]

## Технологии

- Python 3.x
- Tkinter / ttk
- [ttkbootstrap](https://ttkbootstrap.readthedocs.io/en/latest/)[web:207]
- SQLite + FTS5
- PyInstaller (сборка в `.exe`)[web:343]
- Inno Setup (Windows‑инсталятор)[web:427][web:430]

## Установка (из исходников)

1. Клонируйте репозиторий или скопируйте файлы проекта:

   ```bash
   git clone https://github.com/SH3rakhH/Noteproject
   cd zeero-notes
   ```

2. Установите зависимости:

   ```bash
   pip install ttkbootstrap
   ```

3. Убедитесь, что в системе есть SQLite с поддержкой FTS5 (обычно есть в стандартной сборке Python).[web:343]

## Запуск

Запустить приложение можно напрямую из Python:

```bash
python note.py
```

При первом запуске:

- создаётся база `notes.db` и необходимые таблицы;
- создаётся файл PIN (`pin.cfg`) со значением `1234` (по умолчанию);
- при старте показывается окно авторизации — введите PIN и получите доступ к заметкам.

## Структура интерфейса

- Верхняя панель:
  - заголовок «Zeero заметки»;
  - переключатель светлой/тёмной темы.
- Левая колонка:
  - блок «Поиск и фильтр» (FTS‑поле, поиск по тегам, фильтр по категориям, сортировка);
  - список заметок с пагинацией (кнопки «Предыдущая» / «Следующая», индикатор страницы);
  - кнопки «Новая», «Сохранить», «Удалить».
- Правая колонка:
  - заголовок заметки и выбор категории;
  - поле ввода тегов (через запятую);
  - текстовое поле заметки;
  - блок дат «Создано» / «Изменено»;
  - статус‑бар с текущей версией приложения.

## Работа с тегами и поиском

- В поле «Теги (через запятую)» можно указать несколько тегов для заметки.
- В блоке «Поиск по тегам» вводятся теги (через запятую), после чего кнопка «Фильтр по тегам» показывает все заметки с любым из выбранных тегов.
- FTS‑поиск использует виртуальную таблицу `notes_fts` для быстрого поиска по заголовку и содержимому; при проблемах с FTS5 приложение выводит предупреждение и предлагает использовать обычный поиск.[web:343]

## История версий

- При каждом сохранении существующей заметки её предыдущая версия записывается в таблицу `note_versions`.
- В меню Tools → «История версий заметки» можно открыть диалог, выбрать версию по дате и просмотреть её содержимое.

## Экспорт

- **Экспорт БД** — копирование `notes.db` в выбранное место.
- **Экспорт заметки в .txt / .md** — выгрузка текущей заметки в текстовый или Markdown‑файл.
- **Экспорт всех заметок в HTML** — создание одного HTML‑файла с красиво оформленными карточками всех заметок (категория, даты, теги, содержимое).

## Сборка в .exe (PyInstaller)

Минимальная команда (из папки с `note.py` и `version.txt`):

```bash
pyinstaller --noconsole --onefile \
  --name "Zeero заметки" \
  --icon icon.ico \
  --version-file version.txt \
  --collect-all ttkbootstrap \
  note.py
```

- `--collect-all ttkbootstrap` — включает темы, JSON‑файлы и шрифты `ttkbootstrap` (например, `bootstrap.ttf`) в exe, что устраняет ошибки FileNotFoundError.[web:418][web:421][web:425]
- `--version-file version.txt` добавляет информацию о версии в свойства exe (FileVersion, ProductVersion).[web:410][web:415][web:417]

Готовый файл появится в папке `dist` как `Zeero заметки.exe`.

## Создание инсталлятора (Inno Setup)

1. Соберите exe через PyInstaller (см. раздел выше).[web:343][web:431]
2. Установите Inno Setup и создайте скрипт через мастера:
   - Application name: `Zeero заметки`
   - Application version: `1.4.0`
   - Application main executable: `dist\Zeero заметки.exe`.[web:427][web:429]
3. Добавьте при необходимости `notes.db` и другие ресурсы в раздел Files.
4. Скомпилируйте скрипт — будет создан установщик `ZeeroSetup.exe`, который:
   - устанавливает программу в `Program Files`;
   - создаёт ярлыки на рабочем столе/в меню Пуск.[web:427][web:430]
