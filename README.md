# Notes App (Python, Tkinter, SQLite, ttkbootstrap)

Простое десктопное приложение для заметок на Python с современным графическим интерфейсом на Tkinter + ttkbootstrap и базой данных SQLite.  
Приложение позволяет создавать, редактировать и удалять заметки, хранить их в базе, видеть даты создания и изменения, получать всплывающие уведомления и переключаться между светлой и тёмной темой.

---

## 📝 Функционал

- Графический интерфейс с использованием **ttkbootstrap** (современные темы и стили).[web:138][web:143]  
- Хранение заметок в локальной базе **SQLite** (`notes.db`).[web:142]  
- Поля заметки:
  - заголовок,
  - текст,
  - дата создания (`created_at`),
  - дата изменения (`updated_at`).  
- Список заметок слева, выбранная заметка (текст + даты) справа.  
- Кнопки:
  - **Новая** — создание новой заметки,
  - **Сохранить** — сохранение / обновление заметки,
  - **Удалить** — удаление заметки с подтверждением.  
- Всплывающие «toast»‑уведомления при сохранении, обновлении и удалении заметок.  
- Статус‑бар внизу с текстом последней операции и версией приложения.  
- Переключатель между **светлой** и **тёмной** темой (на базе тем `flatly` и `darkly` из ttkbootstrap).[web:129][web:131]  
- Поддержка пользовательской иконки окна (`icon.ico` или `icon.png`).

## 🧰 Технологии

- **Язык:** Python 3.x  
- **GUI:** Tkinter, ttkbootstrap (`ttk.Window`, `bootstyle`) [web:138][web:143]  
- **База данных:** SQLite (`sqlite3`)  
- **Платформы:** Windows, Linux, macOS (зависит от установленного Python и Tkinter)

## 📦 Установка

1. Установите **Python 3.x** (если ещё не установлен).  
2. Склонируйте репозиторий или скачайте архив:

   ```bash
   git clone https://github.com/SherakHh/Noteproject.git
   cd Noteproject
   ```

3. Установите библиотеку `ttkbootstrap`:

   ```bash
   pip install ttkbootstrap
   ```

4. Убедитесь, что в папке есть основной файл приложения, например:

   - `note.py` или `app.py` (название можешь взять из своего кода).

5. (Опционально) положите рядом с `.py` файлом иконку:

   - `icon.ico` — для Windows (предпочтительно),  
   - или `icon.png` — для других платформ.

---

## 🚀 Запуск

В папке проекта выполните:

### Windows

```bash
python note.py
```

или:

```bash
py note.py
```

### Linux / macOS

```bash
python3 note.py
```

При первом запуске автоматически создаётся файл базы данных `notes.db` с таблицей `notes`.

## 💡 Использование

- В верхней части окна показывается название приложения и **переключатель темы**:
  - чекбокс «Тёмная тема» переключает окно между светлой (`flatly`) и тёмной (`darkly`) темами ttkbootstrap.[web:129][web:131]  
- В левой части окна отображается список заметок по заголовкам.  
- Нажмите **«Новая»**, чтобы очистить поля и начать создание новой заметки.  
- Введите заголовок в поле **«Заголовок»**.  
- Введите текст заметки в поле **«Текст заметки»**.  
- Нажмите **«Сохранить»**:
  - если заметка новая — она добавляется в базу,
  - если заметка уже выбрана — её данные обновляются.  
- После сохранения/обновления:
  - появляется небольшое всплывающее уведомление (toast),
  - статус‑бар обновляется сообщением об операции.  
- Выберите любую заметку из списка слева, чтобы:
  - посмотреть/изменить её текст,
  - увидеть даты **создания** и **последнего изменения**.  
- Нажмите **«Удалить»**, чтобы удалить выбранную заметку (перед удалением появляется окно подтверждения).  
- Внизу окна отображается строка статуса вида:

  ```text
  Заметка создана | Версия: 1.3.0 (SQLite + ttk + dates + icon)
  ```

## 🗃 Структура БД

Используется одна таблица `notes`:

- `id` – `INTEGER PRIMARY KEY AUTOINCREMENT`  
- `title` – `TEXT NOT NULL` — заголовок заметки  
- `content` – `TEXT NOT NULL` — текст заметки  
- `created_at` – `TEXT NOT NULL DEFAULT current_timestamp` — дата/время создания (UTC)[web:21][web:55]  
- `updated_at` – `TEXT NOT NULL DEFAULT current_timestamp` — дата/время последнего изменения (UTC)[web:21][web:57]  

При создании заметки `created_at` и `updated_at` заполняются автоматически.  
При обновлении заметки поле `updated_at` устанавливается в текущий момент.

## 🎨 Темы (Light/Dark)

Для оформления используются стандартные темы ttkbootstrap:[web:129][web:143]

- **Светлая тема:** `flatly`  
- **Тёмная тема:** `darkly`  

Переключатель темы реализован через `Checkbutton` и метод `theme_use()` у `root.style`.  
При переключении обновляется статус:

- «Включена светлая тема»  
- «Включена тёмная тема»
